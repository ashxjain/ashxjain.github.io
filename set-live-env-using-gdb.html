Modify environment variables of a running process

Posted on Wed 05 September 2018 in linux

Use GDB to attach to the process

# gdb -p <Process-PID>

Set environment variable

(gdb) p setenv("Key", "Value")

Exit

(gdb) q

Sample C Program

/*
 * setlive_env.c
 */
#include <stdlib.h>
#include <stdio.h>

int
main(int argc, char *argv[])
{
        int     i = 0;
        char    *env_val;

        for (i = 0; i < 10; i++) {
                env_val = getenv("Env_Key");
                printf("Env Val for TZ: %s\n", env_val);
                sleep(5);
        }

        return (0);
}

Build the code:

# gcc setenv_live.c -o setenv_live.o

Run the code:

# ./setenv_live.o

Initial Output:

# ./setenv_live.o
Env Val for TZ: (null)
Env Val for TZ: (null)

Run gdb to attach to the process

# gdb -p <PID-OF-ABOVE-PROCESS>

Set env value and exit

(gdb) p setenv("Env_Key", "New_Value")
